Let \(S\) be a set with \(\lvert S \rvert > 1\), and define a binary operation on
\(S\) by \(xy = y\) for all \(x,y \in S\).
For all \(x,y,z \in S\),
\[
(xy)z = z = x(yz),
\]
and hence the operation is associative; thus \(S\) is a semigroup.

Fix \(a \in S\).
Since \(ax = x\) for all \(x \in S\), the element \(a\) is a left identity.
Moreover, since \(xa = a\) for all \(x \in S\), every element \(x \in S\) has
a right inverse, namely \(a\).

However, for any \(x \in S\) with \(x \neq a\),
\[
x = ax \neq xa = a.
\]
Therefore, \(S\) is not a group.
