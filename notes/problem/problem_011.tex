The following conditions on a group \(G\) are equivalent:
\begin{itemize}
    \item[(i)] \(G\) is abelian;
    \item[(ii)] $(ab)^2 = a^2b^2$ for all $a,b\in G$;
    \item[(iii)] $(ab)^{-1} = a^{-1}b^{-1}$ for all $a, b\in G$;
    \item[(iv)] $(ab)^n = a^nb^n$ for all $n\in \mathbb{Z}$ and all $a,b\in G$;
    \item[(v)] $(ab)^n = a^nb^n$ for three consecutive integers $n$ and all $a, b\in G$.
\end{itemize}
Show (v) $\Rightarrow$ (i) is false if ``three'' is replaced by ``two''.
